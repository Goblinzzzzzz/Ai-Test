# 产品需求文档（PRD）

## 产品概述
- 名称：职场 AI 效能潜力测评（含全员统计）
- 目标：以 10 题评估个人在 5 个维度的 AI 应用潜力，并通过 Supabase 汇总培训班全员数据，输出可视化与综合分析。
- 交付：单文件 HTML（Tailwind CDN + Chart.js CDN + 原生 JS + Supabase 集成）。

## 用户与场景
- 学员：完成测评，查看个人画像、总分、维度雷达图与建议。
- 讲师/班主任：查看班级（cohort）统计（均值、分布、趋势），导出数据做教学复盘。

## 技术栈与形态
- 前端：HTML5 + Tailwind（CDN）+ Chart.js（CDN）+ Vanilla JS。
- 数据：Supabase（开启 RLS）。
- 单文件运行：静态托管或本地直接打开即可使用。

## 前端 UI/UX
- 欢迎页面：标题“职场AI效能潜力测评”，简短說明，有一個明显的「開始测评」按鈕。
- 答案页： 每次显示一道题，或者单页显示所有题目（请设计得容易手机点击）。
- 結果頁（重點）：大字展示学员的画像名称（如：流程设计师）和总分。
- 核心功能：使用Chart.js绘制一个五维雷达图（雷达图），展示用户在上述5个维度（卷入度、指令力、场景力、创新力、技术亲和度）的具体得分。每个维度的满分是6分。底部有一个“重新测试”按钮。
- 风格：移动优先；桌面统计页双列布局；使用尾风CSS（通过CDN导入）实现响应式布局，手机端和电脑端都要美观。
- UI：设计语言：干净的卡片式界面，居中布局，圆角+柔和阴影，移动端优先，基于 Tailwind 实现响应式样式。主题与配色：紫/蓝渐变作为主品牌色，顶边渐变条 gradient-top 提供视觉识别；按钮与进度条统一使用紫→靛的渐变填充。欢迎页：圆形渐变徽标+白卡内容区，主标题加粗、说明文字中灰、胶囊主按钮。测评页：顶部“问题X/10”进度与细进度条、右侧维度标签（浅紫边框+紫色文字），题目大字左对齐；选项为卡片式，左侧 A/B/C/D 圆徽，选中态紫色边框+淡紫底。结果与统计页：图表与按钮采用紫色主题；雷达图填充/描边为紫系，信息卡用浅色系区分。交互反馈：淡色悬停、高亮选中、渐入动画 fade-in；统一圆角为 rounded-xl/rounded-full 强化柔和感。技术栈： Tailwind CDN + Chart.js + 原生 JS；样式通过类组合完成，无额外CSS框架；主题色以紫/靛为主并保持一致性。

## Supabase 集成
- 表：`ai_assessments`
- 字段：`id uuid`、`name text`、`cohort text not null`、`total int not null`、`title text not null`、`d1..d5 int not null`、`answers jsonb not null`、`created_at timestamptz default now()`、`user_agent text`
- RLS：
  - Insert 允许匿名；校验 `total<=30`、`d1..d5<=6`
  - Select：匿名只读全表或只读聚合视图（推荐建立 `ai_assessment_public_stats`）
- 前端常量：`SUPABASE_URL`、`SUPABASE_ANON_KEY`、`DEFAULT_COHORT`、`USE_LOCAL_ONLY`
- 失败兜底：写入失败入本地队列并重试；统计页显示连接状态

## 统计与聚合
- 拉取：`select('*').eq('cohort', selectedCohort)`
- 指标：人数、平均总分、最高/最低、维度均值与标准差、画像分布；趋势线（按日期的平均总分）
- 可视化：Chart.js 雷达/柱状/折线；深色主题与响应式布局
- 导出：CSV（原始记录或聚合摘要）

## 非功能需求
- 性能：惰性渲染；分页或时间窗口；图表最小高度 320–360px
- 可用性：触控区域大；键盘可操作；颜色对比良好
- 隐私：不采集敏感数据；昵称可选且默认匿名显示
